Some generic data structures in C.

Depends on Cutils.h which is a collection of
QOL tools.

## Features
- Typesafe Dynamic arrays
    - Declared with macros
    - Type safe
    - Deletion
    - Insertion
    - Appending
    - Resize
    - Reserve
    - Custom allocator support via Cutils.h
- Typesafe Generic Dynamic Hashmaps
    - Declared with macros
    - type safe
    - Insertion
    - Deletion
    - Lookups
    - Reserve
    - Clear
